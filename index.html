Projects

I have developed many school, extracurricular, and personal projects. Here are some notable ones:

  • Apogee Control System: An actively controlled air brake system for a high-power rocket.
  • iPatch: A macOS GUI tool to inject dynamic libraries into iOS applications (enabling jailed tweaks).
  • Cubik / Opticube: Software library (Swift) and CLI application (C) to model and solve the 3x3 Rubik's cube.
  • B-Minor Compiler: A compiler for a C-like programming language that generates executable x86 assembly.
  • ifunnyapi: A reverse-engineered (using man-in-the-middle tools) social media API wrapper on PyPI.
  • Reddit Chronology: A bot to scrape Reddit threads and generate YouTube videos with text-to-speech.
  • Maria: An artifical intelligence model to play original Super Mario Bros. levels.
  • Atlas: A mobile social media application to track and share fitness experiences.
  • Unessay: A recipe search engine given the ingredients in your kitchen (History of Food class project).
